ASTRA Banner - 纯HTML版本

这是您的ASTRA Banner的纯HTML/CSS版本，包含了完整的动画效果和渐变样式。 可以直接复制到任何HTML文件中使用，无需React或其他框架。

ASTRA

如何在您的HTML文件中使用：

1. 最简单的使用方法：

复制上面整个Banner的HTML和CSS代码到您的HTML文件中即可。

2. 仅使用Banner部分：

<!-- 只复制这部分到您的HTML中 --> <div class="astra-banner"> <div class="banner-background"> <div class="gradient-circle-1"></div> <div class="gradient-circle-2"></div> <svg class="geometric-lines" viewBox="0 0 400 200"> <defs> <linearGradient id="lineGradient" x1="0%" y1="0%" x2="100%" y2="100%"> <stop offset="0%" stop-color="#87ceeb" /> <stop offset="100%" stop-color="#483d8b" /> </linearGradient> </defs> <path class="line-path" d="M0,100 Q100,50 200,100 T400,100" stroke-width="2" opacity="0.15"/> <path class="line-path-2" d="M0,120 Q150,70 300,120 T400,120" stroke-width="1" opacity="0.1"/> </svg> <div class="grid-pattern"></div> </div> <div class="banner-content"> <h1 class="astra-text">ASTRA</h1> </div> <div class="glow-overlay"></div> </div>

3. 自定义集成：

/* 将CSS样式添加到您的样式表中 */ /* 然后使用上面的HTML结构 */ /* 可以修改 .astra-banner 的尺寸和位置 */ .my-custom-container .astra-banner { max-width: 800px; margin: 2rem auto; }

4. 添加点击事件：

<script> function handleBannerClick() { // 您的点击处理逻辑 console.log('Banner被点击了！'); // 例如：跳转到其他页面 // window.location.href = '/about'; } </script>

这个HTML版本包含了所有的视觉效果：渐变背景、动画文字、悬停效果和几何装饰。 您可以根据需要调整颜色、尺寸或动画效果。
